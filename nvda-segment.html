NVIDIA Datacenter Segment Split — Local AI 是否在跟上 Hyperscaler？

追蹤 NVIDIA Datacenter 營收的 5 大客戶 segment 比例變化（2023-2027F）：🟦 CSP/Hyperscaler、🟪 Consumer Internet（xAI/Tesla/TikTok）、🟧 Enterprise on-prem（MGX 主場）、🟥 Sovereign AI、🟢 Neocloud。核心問答：Q: Local AI（Enterprise + Sovereign）是否在跟上？ → Yes，明確。Local AI 占比從 2023 15% → 2027F 36%（翻倍以上），與 CSP gap 從 50ppt → 7ppt（即將平分秋色）。Sovereign AI YoY +57%、Enterprise +52%是 2027 最快兩個 segment（vs CSP +25%）。MGX modular reference architecture 是 enabler — Foxconn/Quanta/Wistron/Asus/Supermicro 等 ODM 出貨給 mid-size enterprise + sovereign。

Methodology & Disclaimer

5 Segment 定義

  • 🟦 CSP/Hyperscaler：Microsoft / Google / Amazon / Meta / Oracle 等大型雲端服務商
  • 🟪 Consumer Internet：xAI (Memphis Colossus) + Tesla + TikTok + 其他 social media
  • 🟧 Enterprise on-prem：銀行 + 製造 + 醫療 + 電信 + 公部門 + 國防 — **本地 AI server，MGX 主場**
  • 🟥 Sovereign AI：國家級 AI infra（Saudi Humain $20B+ / UAE / Japan SoftBank Stargate / India Yotta+Reliance / S Korea Naver / 歐洲 sovereign cloud）
  • 🟢 Neocloud：CoreWeave / Lambda / Crusoe / Together / Fireworks 等 GPU-only cloud

用戶問題的 Three-layer Answer

  • Q1: Local AI 是否跟上 Hyperscaler？
  • A1: Yes — Local AI 占比 15% (2023) → 36% (2027F)，gap 從 50ppt 縮至 7ppt。但仍未超過 CSP（CSP 2027F 仍 43%）。
  • Q2: 速度有多快？
  • A2: 快。2027F YoY growth: Sovereign +57% / Enterprise +52% / Neocloud +36% / Consumer +30% / **CSP +25%（最慢）**。Local AI 是 NVIDIA 未來 3 年最大成長 driver。
  • Q3: MGX 角色？
  • A3: MGX 是 enabler。Modular GPU eXtensible reference design (GB200 NVL2/NVL36)，Foxconn/Quanta/Wistron/Asus/Supermicro 等 ODM 出貨。為 mid-size enterprise + sovereign 設計（不是 hyperscaler 大客戶 — 大客戶有自己的 custom rack design）。

關鍵 spotlights

  • NVDA Datacenter 2025 ~$200B → 2027F $380B（5 年 8x 從 2023 $47.5B）
  • 2027F segment 排名：CSP $163B > Consumer $61B > Enterprise $84B > Sovereign $53B > Neocloud $19B
  • Sovereign AI 訂單列表（已揭露）：Saudi Humain $20B+ / UAE $10B+ / Japan SoftBank / India Yotta + Reliance Jio / S Korea Naver+SK Hynix / 歐洲多國 sovereign cloud
  • Enterprise on-prem 加速因素：data sovereignty + AI privacy compliance + cost predictability + regulatory（特別是金融/醫療/政府）。Hyperscaler 服務雖便利但 enterprise 需 fully-on-prem 為主
  • MGX OEM/ODM 受益鏈：Foxconn 2317 / Quanta 2382 / Wistron 3231 / Supermicro / Dell 等。MGX 量產 = 台廠 ODM 直接受惠（cross-link Rack Line (TW)）

與其他 page 的對話

  • NV GPU Market：Blackwell backlog 3.6M / GB200 出貨 = 滿足 5 segment 合計需求
  • Lab Compute (GW)：frontier lab GW 部分屬於 CSP segment（OpenAI = MSFT/Oracle CSP）+ 部分屬於 Consumer Internet（xAI 自建）
  • CSP/Neocloud Capex：CSP 占 NVDA 50% 對應 hyperscaler $300B+ capex 主要採購 NVIDIA
  • Rack Line (TW)：MGX 出貨 = 台廠 ODM (鴻海/廣達/緯創) 直接受惠的 segment
  • Country Compute Race：Sovereign AI segment 對應各國家算力建設
