Methodology & Disclaimer

換算 framework：1 GW ≈ 500K GPU 等效

  • 1 GW datacenter 平均含 ~500K GPU（H100/H200/B200 等效）
  • 含 cooling overhead (~20%) + networking + storage + 待機 buffer
  • 實際密度依 chip 世代、liquid cooling 滲透率而異 (Rubin 時代每 GPU 功耗更高，密度可能下降)
  • 10 GW = 約 500 萬 GPU 等效（user-defined Anthropic 2030 vision）

6 Lab + Chip Mix 戰略對照

  • OpenAI（NVIDIA 70 + Maia 20 + AMD 10）：Stargate $500B 五年計劃，倚重 Microsoft Azure + Oracle + Crusoe 多方融資
  • Anthropic（Trainium 50 + TPU 30 + NVIDIA 20）：**最 diversify**，Amazon $8B + Google $2B 雙投資推升 chip mix。訓練成本 4x 比 OpenAI 便宜（公開揭露）
  • Google DeepMind（100% TPU）：垂直整合 11 年累計優勢，Gemini 全系列 100% TPU 訓練
  • Meta AI（NVIDIA 80 + MTIA 15 + AMD 5）：NVIDIA 主力，MTIA 主要做 recommendation/inference 降本
  • xAI（100% NVIDIA）：純自建 cluster + 速度優先（Memphis Colossus 122 天 0→100K H100）
  • DeepSeek（NVIDIA 50 + 國產 50）：受出口管制，靠 algorithm efficiency（V3 訓練 $5.6M）部分補償硬體劣勢

2030 vision 警告

  • 2030 vision 為 D 級（公司公開承諾或 capex commitment 推算，非 hard guidance）
  • 合計 ~70 GW = 3,500 萬 GPU 等效，需問：**市場真能消化？**
  • 對照 LLM Market ARR 趨勢：若 2030 ARR ~$300B 是否可支撐 70 GW capex 攤銷？需 ARR / GW ratio 觀察
  • Cross-link LLM Market：滲透率 ceiling + ARR ceiling 是 70 GW vision 的最大不確定因素

與其他 page 的對話

  • CSP / Neocloud Capex：lab compute 是 capex 投入後的「實體 footprint」（lab demand → cloud capex → chip orders）
  • NV GPU Market：6 lab 2030 NVDA 占 65% × 35M GPU = ~22M NVDA GPU vs Blackwell backlog 3.6M = 6 倍 long-term demand
  • AI Chip Alt：Trainium 50% Anthropic + TPU 100% Google + Maia 20% OpenAI = in-house ASIC 結構性增長 driver
  • Country Compute Race：6 lab 全 US except DeepSeek，US frontier 主導但中國 algorithm 反擊
