Methodology & Disclaimer

三類替代品分類

  • 對外銷售（與 NVIDIA 直接競爭）：AMD MI300/MI325/MI355、Intel Gaudi 3/Falcon Shores
  • In-house ASIC + Cloud 對外：Google TPU（Vertex AI）、AWS Trainium（Bedrock）、Microsoft Maia（Azure）
  • In-house ASIC 純自用：Meta MTIA（Reels/Llama 推論，不對外）、Apple ANE（邊緣，本頁未收）

關鍵 insights

  • NVIDIA 護城河：CUDA 生態 + 軟硬整合 + frontier model 訓練平台。市占小幅縮小（88→80%）但絕對量級持續快速成長。
  • AMD 是唯一規模化 commercial 替代：MI325X HBM3e 256GB（vs H200 141GB）在 LLM inference memory-bound workload 1.6x 領先 H200。客戶含 Microsoft Azure、Meta、Oracle、Dell。
  • Hyperscaler 自家 ASIC 4x 翻倍：2024 ~2% → 2026 ~8%。Google TPU v8 Ironwood + AWS Trainium 3 + Microsoft Maia 200 + Meta MTIA Santa Barbara（廣達 ODM 6,000 rack）皆 2025-2026 量產。
  • Anthropic + AWS Project Rainier：400K Trainium 2 cluster 訓練 Claude，是非 NVIDIA 訓練前沿 model 的最大規模實驗。
  • Intel Gaudi 邊緣化：2024 launch Gaudi 3 號稱 vs H100 1.5x 性能/-50% 價格，但生態系敗、CUDA lock-in，2025 全年僅 ~$500M 營收（vs NVIDIA H100 ~$50B+）。

For investors / analysts

  • Watch：(1) AMD MI400 (2026 H2) 性能 vs B200，能否拿到 first 10% 市占；(2) TPU v8 Ironwood 量產進度（影響 Google AI infra 內部成本）；(3) AWS Project Rainier 完成度（Anthropic Claude 訓練 vs OpenAI GPT-5 的 cost 對比）。
  • Risk to NVIDIA thesis：若 in-house ASIC 從 8% (2026) → 20%+ (2028)，NVIDIA datacenter 成長率將顯著減速（市占 + 絕對量級雙縮）。
