Methodology & Disclaimer

Grade 分級

  • Grade A — 一手公開（法說會、MOPS 公告、年報、SEC filing、OCP 產品註冊）
  • Grade B — Tier-1 媒體深度（Bloomberg、Reuters、日經、工商時報、經濟日報、Digitimes、SemiAnalysis、TrendForce）
  • Grade C — 多源交叉但無一手（行業會議共識、多家券商重複敘述）
  • Grade D — 單一來源或分析師推估；本頁 D 級項目皆附 Bear/Base/Bull 三情境於 notes 欄位

Rack ODM 揭露限制（重要）

  • 四家 rack ODM（Foxconn / Quanta / Wiwynn / Inventec）皆採公司整體或 AI 伺服器整體營收揭露，不分項至單一 DC 客戶
  • 本頁所有 shipment 類別 (rack_assembly / l6_server / l10_server / l11_rack / backend_network) 的 est_revenue_twd_mn 一律為 null（不可填，避免製造虛假精確度）
  • Stat 3「2025E AI Server Revenue (Aggregate)」採以下 per-supplier 估算加總：
    • 2317 Foxconn NT$1.0 兆 — 2025 前三季法說會（劉揚偉 2025/11/12）AI server 營收破兆元（Grade A）
    • 2382 Quanta NT$1.0 兆 — 2025 全年法說會 AI server 略超 1 兆，YoY +100%+（Grade A）
    • 6669 Wiwynn NT$0.475 兆 — 2025 全年合併營收 NT$950.66B × AI 占比過半 ~50%（Digitimes 2026/02/27 + estimate）
    • 2356 Inventec NT$0.20 兆 — 2025E AI server ~NT$200B (33% of total revenue)（Grade C 自有推算）
    • 合計 ≈ NT$2.675 兆，僅在「全選」filter 下顯示完整加總；單一供應商 filter 顯示對應供應商項
  • Wiwynn 與 Inventec 的 AI 占比為 estimate；公司未獨立揭露 AI server 全年絕對金額。Wiwynn 官方表示「AI 占比過半」、Inventec 官方表示「Q4 AI server 超過 server 業務 50%」

Rack ODM 客戶集中度（公開揭露）

  • Foxconn — NVIDIA GB200 NVL72 全球單月最大 rack 出貨商（2025/04 估市占 40%，TweakTown / TrendForce 引用）；客戶含 Oracle / CoreWeave / Meta / AWS
  • Quanta — NVIDIA GB200 全球第二（市占 ~30%）；客戶含 Meta / Google / Microsoft（公司年報三大客戶各占 ~30% / 25% / 15%）
  • Wiwynn — 2024 三大客戶占營收 49% / 28% / 21%（合計 98%），業界共識為 Meta（#1）/ Microsoft / AWS（B 級 Silba 引用 TWSE 年報）
  • Inventec — Tier-2 ODM；AI server 客戶含 Microsoft / Meta（白牌）+ NVIDIA H100/H200/B200 platform；公司未公開揭露三大客戶名單
