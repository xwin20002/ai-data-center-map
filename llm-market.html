Methodology & Disclaimer

Schema (phase5-llm-v1)

  • 每行 = 「某 provider 在某年某 metric 的數值」。schema 與 phase 3/4 supply chain 不同（provider 取代 supplier_ticker，year 取代 dc_project，多種 unit）
  • Categories：ARR_total / ARR_B2B / ARR_B2C / MAU_consumer / paying_subscribers / market_share_consumer / market_share_enterprise / model_release / valuation / compute_capex / enterprise_customers / api_requests / killer_app
  • Units：USD_mn（百萬美元）、M_users（百萬用戶）、percent、USD_per_month、businesses、B_per_month、models

Grade 分級

  • A — 公司直接揭露（CEO/CFO 公開聲明、官方新聞稿、Series F/G 投資揭露）
  • B — Tier-1 媒體（TechCrunch / Bloomberg / The Information）+ sell-side research summary
  • C — 多源券商共識，特別是 Google Gemini-attributable revenue（Google 不獨立揭露）
  • D — 2027 forecast 含 Bear/Base/Bull 三情境

關鍵 Insights

  • Anthropic 2026 Q1 反超：ARR $30B (Mar 2026) vs OpenAI $25B (Mar 2026)。Claude Code $2.5B ARR 為主要 driver。1,000+ enterprise 年付 >$1M。
  • OpenAI 仍領先 consumer 市場：900M weekly active users (Mar 2026) vs Gemini 750M MAU vs Claude 50M MAU。65% chatbot traffic share。
  • Google Gemini 暴增：Consumer 市占從 5.7% (Jan 2025) → 21.5% (Jan 2026)，4x 成長。Search / Workspace 整合是核心 distribution 優勢。
  • B2B vs B2C 結構差異：Anthropic 90% B2B（enterprise-first）vs OpenAI 60% B2B / 40% B2C（consumer-strong）vs Google 80% B2B。
  • Compute capex 軍備競賽：OpenAI 2025 ~$11B / Stargate $500B 五年計劃；Anthropic 4x cheaper training（vs OpenAI）；Google 自家 TPU 垂直整合 cost advantage。

產地分佈

  • 三家 anchor 全部 🇺🇸 美國 HQ（San Francisco / Mountain View）
  • 缺席的 vendors（後續可加）：xAI Grok（🇺🇸 私有）、Meta Llama（🇺🇸 NASDAQ 開源）、DeepSeek（🇨🇳 私有）、Mistral（🇫🇷 EU 私有）、Cohere（🇨🇦 CA 私有）、Alibaba Qwen（🇨🇳 NYSE: BABA）
  • 地緣政治分布：US 三巨頭 + CN 雙巨頭（DeepSeek + Qwen）+ EU/CA 補位（Mistral / Cohere）

2027 Forecast（D 級 Bear/Base/Bull）

  • OpenAI 2027：Bear $35B（Anthropic 進一步分食）/ Base $50B（Stargate 兌現 + 雙位數 Hyperscaler 簽約）/ Bull $80B+（AGI 商業化 + Sora 訂閱爆發）
  • Anthropic 2027：Bear $50B（成長收斂至 +60-80%）/ Base $80B（持續 Hyperscaler 簽約 + Code 滲透）/ Bull $120B+（Reasoning + agentic 取代多 SaaS 工具）
  • Google 2027 Gemini-attributable：Bear $20B（Apple Search 整合受阻）/ Base $30B（Workspace AI + Cloud Vertex growth）/ Bull $50B+（Search AI 直接 monetization）
