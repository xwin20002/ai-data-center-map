Methodology & Disclaimer

Phase 4 Multi-Currency Framework

  • Phase 4 vendors 跨 SZSE / NYSE / NASDAQ / TWSE，schema 含 hq_listed / home_country / currency / est_revenue_local_mn / est_revenue_twd_mn
  • FX rate (lock 2026-04-26)：1 USD = 31 TWD；1 CNY = 4.4 TWD（用於 cross-vendor 統一比較）
  • Stat 3 採 per-supplier hardcoded 2025E AI optical revenue（TWD-equivalent 加總）

Grade 分級

  • Grade A — 一手公開（10-K/10-Q SEC filing、SZSE 季報、法說會逐字稿、官方新聞稿）
  • Grade B — Tier-1 媒體 + sell-side research（Bloomberg / Reuters / 日經 / Digitimes / SemiAnalysis / Cignal AI / Goldman Sachs / Daiwa / KGI）
  • Grade C — 多源券商共識，無一手
  • Grade D — 單一來源或 scenario estimate；本頁 D 級項目皆附 Bear/Base/Bull 三情境

Stat 3 算法（per-supplier hardcoded TWD mn）

  • 🇨🇳 300308 Innolight — NT$1,648 億（光通訊模組 RMB 374.57 億 × 4.4 = TWD 164,811 mn，FY2025 Grade A SZSE 揭露）
  • 🇺🇸 COHR Coherent — NT$775 億（FY2025 AI Datacenter exposure USD 2,500M × 31 = TWD 77,500 mn，Grade B 分析師整理）
  • 合計 ≈ NT$2,423 億，僅在「全選」filter 下顯示完整加總；單一 vendor filter 顯示對應項

中美雙 anchor 競合定位

  • Innolight 主項：800G/1.6T transceiver 模組規模領先；NVIDIA Mellanox + Hyperscaler 直供；ASP 較美廠低 15-25%（製造成本優勢）
  • Coherent 主項：垂直整合 InP wafer (6-inch Texas + Sweden) → EML laser chip → transceiver 三層；NVIDIA + Cisco + Arista + EML chip 上游賣給 Innolight/Eoptolink
  • 競合既鬥又合：Coherent 賣 EML laser chip 給 Innolight → Innolight 模組成功 = Coherent 上游 chip 收入；同時兩家在 transceiver 終端產品競爭
  • 1.6T 早期市場：Innolight 50-70% 領先（B 級 SemiAnalysis / Goldman Sachs 共識），Coherent 美廠 anchor ~15-20%
